# RainRoute (web)

RainRoute 的瀏覽器版：輸入出發地與目的地，查詢騎車路線是否會淋雨。

## 開發

```bash
VITE_BACKEND_URL=http://localhost:8000 npm run dev
```

## 建置

```bash
npm run build
```

## 測試

```bash
npm test
```

## 環境變數

| 變數 | 說明 | 預設值 |
| --- | --- | --- |
| `VITE_BACKEND_URL` | 後端 API base URL | `http://localhost:8000` |
